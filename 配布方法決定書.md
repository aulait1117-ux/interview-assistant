# 面接アシスタント β配布方法 決定書

作成日：2026-06-30

---

## 決定内容

| コンポーネント | 方式 | 理由 |
|--------------|------|------|
| **バックエンド** | Renderデプロイ | `.env.production`に設定済み、URLビルド済みJS内に埋め込まれている |
| **フロントエンド** | Render静的ホスティング | render.yaml設定済み |
| **Electronアプリ** | .exeインストーラーをローカル配布 | `dist-electron/面接アシスタント Setup 0.1.0.exe` |

---

## アーキテクチャ

```
[ユーザーPC]                          [Render]
┌────────────────────────────┐       ┌──────────────────────────────┐
│ 面接アシスタント.exe        │       │ interview-assistant-hrar     │
│ (Electron + frontend/dist) │──────▶│ FastAPI バックエンド          │
│                            │  API  │ - /api/speech/transcribe     │
│ ・Zoom上にオーバーレイ表示  │       │ - /api/interview/hint-stream │
│ ・getDisplayMediaで音声録音 │       │ - /api/billing/ (Stripe)     │
│ ・バックエンドへ音声送信    │       │ - /api/auth/ (Google OAuth)  │
└────────────────────────────┘       └──────────────────────────────┘
                                              │
                                      [外部API]
                                      - OpenAI Whisper (音声→テキスト)
                                      - Claude API (AI回答生成)
                                      - Stripe (決済)
```

## β配布の前提条件

### Render側（チーム作業）
- [ ] バックエンドのデプロイ完了 (`interview-assistant-hrar.onrender.com`)
- [ ] 環境変数の設定
  - `ANTHROPIC_API_KEY` ← Claude API
  - `OPENAI_API_KEY` ← Whisper API
  - `STRIPE_SECRET_KEY` ← 荒川さん側 (β期間は後回し可)
  - `GOOGLE_CLIENT_ID` ← Google OAuth

### ユーザー側（追加インストール不要）
- Windows 10/11 (64bit)
- インターネット接続
- Zoom（面接時）

## βテスト期間の決済について

- Stripe本番設定は荒川さん側で実施予定（07/07目標）
- β期間中は以下のいずれかで対応：
  - 無料招待（決済スキップ）
  - Stripeテストモードで動作確認のみ

```

├── latest.yml                          ← 自動更新用（将来的に使用）
```
